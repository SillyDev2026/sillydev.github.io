Promise Module

Async Workflow System

A powerful Luau promise implementation for Roblox featuring chaining, retries, event handling, and async utilities.

Get Module Back to Modules

Overview

Core behavior and system design

Promise is an asynchronous control flow system designed for Roblox. It allows structured handling of async tasks without callback hell.

Async Chaining Reliability

Basic Usage

local Promise = require(Promise)

Promise.new(function(resolve)
    task.wait(1)
    resolve("Hello World")
end):andThen(function(result)
    print(result)
end)

Chaining

Promise.new(function(resolve)
    resolve(10)
end)
:andThen(function(v)
    return v * 2
end)
:andThen(function(v)
    print(v) -- 20
end)

Error Handling

Promise.new(function(_, reject)
    reject("Something went wrong")
end)
:catch(function(err)
    warn(err)
end)

Finally

Promise.new(function(resolve)
    resolve("Done")
end)
:finally(function()
    print("Always runs")
end)

Advanced Examples

Utilities and extended API usage

Promise.all

Promise.all({
    Promise.resolve(1),
    Promise.resolve(2),
    Promise.resolve(3)
}):andThen(function(results)
    print(results[1], results[2], results[3])
end)

Retry

Promise.retry(function(resolve, reject)
    if math.random() > 0.5 then
        resolve("Success")
    else
        reject("Failed")
    end
end, 3)

Delay

Promise.delay(2):andThen(function()
    print("2 seconds passed")
end)
