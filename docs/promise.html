Promise Module

Async Workflow System

A Luau Promise implementation with chaining, retries, async handling, and event utilities.

Get Module Back to Modules

Installation

local Promise = require(path.to.Promise)

Basic Usage

local Promise = require(PromiseModule)

Promise.new(function(resolve)
    task.wait(1)
    resolve("Hello World")
end):andThen(function(result)
    print(result)
end)

Chaining

Promise.new(function(resolve)
    resolve(10)
end)
:andThen(function(v)
    return v * 2
end)
:andThen(function(v)
    print(v)
end)

Error Handling

Promise.new(function(_, reject)
    reject("Something went wrong")
end)
:catch(function(err)
    warn(err)
end)

Finally

Promise.new(function(resolve)
    resolve("Done")
end)
:finally(function()
    print("Always runs")
end)
